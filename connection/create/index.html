    Create new Connection

    This section describes how to create a new text connection. Creating other connection types is similar.

    Create New Text Connection

    1. Click New Connection button on the top left corner to create a new connection.

    2. Install text driver if you do not have it (skip to step 9 if you already had one). Click Download and Install to install the required driver.

    3. The progress bar will show the download progress. Once the download completes, the installation will run automatically.

    4. Click Yes to accept the driver installation.

    5. Click Next to continue. ![Welcome Wizard]](/img/connection/new_conn/new_con5.png)

    6. Read the license agreement carefully and check accept box. Click Next to continue.

    7. Select install location and click Install.

    8. Click OK to continue.

    9. Enter Connection Name and click Browse.

    10. Select a text file and click Open.

    11. The preview panel will show the content of your text file. Click Save to save connection. Click Define Custom Format if what you see in the preview is not what you expect.

    12. The newly created connection will show on the Connection section of the Home tab page.

    Create New Excel Connection

    1. Create a new Excel connection is very similar to creating a text connection. Enter Connection Name and click Browse to browse to the file location.

    Create New SQLite Connection

    1. Enter Connection Name and click Browse to browse to the file location. Enter Password if your datatbase file is password-protected.

    Create New PostgreSQL Connection

    1. Enter Connection Name and the PostgreSQL server information. Click Advanced for more options.

    2. Enter Initial Database if you want to set a default database when open the connection. Click Save to complete adding new connection.

    Create New MySQL Connection

    1. Enter Connection Name and the PostgreSQL server information. Click Save to complete adding new connection.

    Create New Microsoft SQLServer Connection

    1. Enter Connection Name and the SQLServer server information. Vytics supports both Windows Authentication and Server Authentication. Click Advanced for more options.

    2. Enter Initial Database if you want to set a default database when open the connection. Click Save to complete adding new connection.
