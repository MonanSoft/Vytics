    Create New Project

    Project is where you manage your data and reports. This section describes basic operations when working on a project.

    Create Project

    1. Click New Project button on the top left corner to create a new project.

    2. Enter Project Name and Project Description. Click Save to complete.

    3. Project Home will automatically display.

    Delete Project

    1. Click Delete Project icon to delete an existing project.

    2. Click Yes to confirm.

    Open Existing Project

    1. Click on project name Home Energy to open the project.

    2. The Project Home will display. There are few important items:

      • (1) New Data button to create a new Data
      • (2) New Report button to create a new Report
      • (3) Project treeview to display all data and reports
      • (4) Snapshots of recent reports.
